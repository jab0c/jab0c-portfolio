# Code of Conduct

Thanks for stopping by. This repo is meant to be welcoming and low stress.

## Our Pledge
We keep things respectful, inclusive, and helpful. Disagreement is fine. Disrespect is not.

## Expected behavior
- Be kind and constructive
- Assume good intent
- Keep feedback about the work, not the person

## Unacceptable behavior
- Harassment, insults, or hate
- Doxxing or sharing private info
- Persistent bad faith arguments

## Enforcement
If you run into an issue, please email: contact@jab0c.com
